Base de datos


La base de datos de MusicaList almacena los datos del sistema como lo son las canciones, las listas y la información de los usuarios independientemente sean administrador o votante. cuenta con seis tablas (admin, cancion, lista, persona, persona_cancion y persona_lista), de las cuales dos de ellas corresponden a tablas intermedias (persona_cancion y persona_lista).
